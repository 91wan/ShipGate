# Security Policy

Please report suspected fail-open behavior, path escape, report leakage, command
injection, unsafe installation, or secret-detection bypass privately to the
repository maintainers before public disclosure.

Include the ShipGate version, operation/source, minimal reproduction, expected
blocking behavior and a redacted report. Do not include live credentials,
private keys, personal paths or confidential repository content.

ShipGate is a configured indicator scanner and publication-structure gate. A
passing result is not a guarantee that a repository contains no secret or
vulnerability. Keep independent credential rotation, code review, signing and
platform release controls in place.
